# 体素电梯（Voxel Elevator）
一款为 Minecraft 打造的全新电梯模组

【图标：暂无】

## 开发计划

### 一、电梯轿厢
- [ ] 通过自定义实体实现（含注册与移动逻辑）
- [ ] 轿厢停止移动后，可恢复为方块形态
- [ ] 支持音效播放

### 二、机房/机箱方块
- [ ] 曳引机：提供多种款式与型号，不同型号具备不同参数
- [ ] 控制柜：用于楼层设置与参数配置

### 三、控制系统
- [ ] 外呼面板固定框：内置默认样式，支持导入自定义样式包
- [ ] 内呼面板固定框：内置默认样式，支持导入自定义样式包
- [ ] 尝试实现可直接交互的内呼面板；若实现成本过高，则开发内呼选层 UI
- [ ] 基站层消防开关

### 四、装饰性物件
- [ ] 到站灯

### 五、物品
- [ ] 司机钥匙
- [ ] 轿厢框架构建器

### 六、井道方块
- [ ] 导轨
- [ ] 楼层标记

### 七、门系统
- [ ] 电梯门：支持不同安装方式
